Developing and Analyzing Algorithms for the Multi-Armed Bandit
Prof. Shivaram Kalyanakrishnan

We developed algorithms for the regular multi-armed bandit setting and a "batch-sampling" setting, in which a batch of pulls must be executed before the rewards of the pulls are received. We have empirically demonstrated improvements over other algorithms for these settings. We have made substantial progress towards formally proving non-asymptotic upper bounds on regret.

Parallel Computing for the Laplace Equation on Unstructured Grids
Prof. Shivasubramanian Gopalakrishnan

The solution to the laplacian equation is used in a large number of domains, including finding the steady state temperatures of an object given the boundary conditions. We developed a solver that iteratively computes the solution to the laplacian for arbitrarily shaped objects making use of whatever number of processors is available. The parallel versions were designed with the MPI paradigm, the problem was divided into smaller parts, distributed among processors and communication was perfomed between the processors every iteration. To minimize the communication volume between processors and achieve maximum speed-up graph partitioning algorithms from METIS were used. With 16 processors available, we were successful in achieving ~11x speedup.
